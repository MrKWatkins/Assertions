# StringAssertionsChain Constructors

Enables chaining of assertions on a string value after a successful assertion.

```c#
public StringAssertionsChain(StringAssertions stringAssertions);
```

## Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| stringAssertions | [StringAssertions](MrKWatkins.Assertions.StringAssertions.md) | The assertions object to chain from. |
